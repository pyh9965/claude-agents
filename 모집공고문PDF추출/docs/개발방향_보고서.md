### Phase 2: Planner 확장 (2단계)

**작업 내용:**
1. planner.md에 PDF 입력 처리 로직 추가
2. 분양가 데이터 → brief.md 변환 템플릿 작성
3. 마케팅 앵글 자동 생성 로직 추가

```markdown
### PDF 입력인 경우 (아파트 모집공고문)

1. pdf-price-extractor 스킬 호출
2. 추출된 분양가 데이터 분석
3. 마케팅 앵글 도출:
   - 가격 경쟁력 (주변 시세 대비)
   - 입지 장점
   - 투자 관점 포인트

4. brief.md에 부동산 섹션 추가
```

## 5. 예상 활용 시나리오

### 시나리오 1: 신규 분양 단지 콘텐츠 자동 생성

**입력:**
```
드파인 연희 모집공고문.pdf
```

**자동 생성 콘텐츠:**

| 플랫폼 | 콘텐츠 제목 |
|--------|------------|
| 블로그 | "드파인 연희 분양가 분석: 59㎡ 11.6억, 연희동 입지 가치는?" |
| 유튜브 | "서대문구 신축 아파트 분양가 총정리 (18개 타입 층별 가격)" |
| X 스레드 | "🏠 드파인 연희 분양가 공개! 59㎡ 11.6억~12.3억..." |
| 쇼츠 | "1분 만에 보는 드파인 연희 분양가 핵심" |
| 링크드인 | "연희동 대장 아파트 등극? 드파인 연희 투자 관점 분석" |

---

### 시나리오 2: 비교 분석 콘텐츠

**입력:**
```
드파인 연희.pdf + 래미안 연희.pdf (복수 PDF)
```

**자동 생성:**
- "연희동 신축 분양가 비교: 드파인 vs 래미안"
- 가격대별 비교표
- 면적별 가성비 분석
