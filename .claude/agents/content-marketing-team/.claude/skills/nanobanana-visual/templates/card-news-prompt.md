# 카드뉴스 프롬프트 템플릿

## 기본 프롬프트 구조

```
Create a professional Instagram carousel slide for {SLIDE_TYPE}.

Style: Clean, modern, minimalist
Size: 1080x1350px (4:5 ratio)
Color scheme: {COLOR_SCHEME}
Font style: Bold sans-serif for titles, regular for body

Content:
- Title: {TITLE}
- Subtitle: {SUBTITLE}
- Body text: {BODY_TEXT}
- Icon/Illustration: {VISUAL_ELEMENT}

Design requirements:
- Mobile-first readability
- High contrast text
- Consistent brand styling
- Adequate white space
```

## 슬라이드별 템플릿

### 슬라이드 1 (표지)

```
Create a captivating Instagram carousel cover slide.

Title: "{MAIN_TITLE}"
Subtitle: "{HOOK_TEXT}"
Visual: {ICON_OR_ILLUSTRATION}

Style notes:
- Eye-catching design that encourages swipe
- Include swipe indicator (arrow or "Swipe →")
- Bold, prominent title
- Intriguing subtitle

Color: {PRIMARY_COLOR} background with {CONTRAST_COLOR} text
```

### 슬라이드 2-6 (본문)

```
Create an informative Instagram carousel body slide.

Point number: {NUMBER}
Main text: "{KEY_POINT}"
Supporting text: "{EXPLANATION}"
Visual element: {RELEVANT_ICON}

Style notes:
- Clear hierarchy (number → main point → explanation)
- One concept per slide
- Easy to scan quickly
- Consistent layout with other slides

Color: Match series style
```

### 마지막 슬라이드 (CTA)

```
Create an engaging Instagram carousel CTA slide.

Main text: "{CTA_MESSAGE}"
Action prompts:
- "저장해두세요"
- "팔로우하기"
- "친구에게 공유"

Visual: {CTA_ICON}

Style notes:
- Clear call-to-action
- Save/Follow/Share icons
- Closing message that resonates
- Brand logo (if applicable)
```

## 변수 설명

| 변수 | 설명 | 예시 |
|------|------|------|
| {SLIDE_TYPE} | 슬라이드 유형 | cover, body, cta |
| {COLOR_SCHEME} | 색상 조합 | "navy blue and white" |
| {MAIN_TITLE} | 메인 제목 | "AI로 월 100만원 벌기" |
| {HOOK_TEXT} | 후킹 문구 | "이것만 알면 됩니다" |
| {KEY_POINT} | 핵심 포인트 | "첫 번째, 틈새시장 찾기" |
| {EXPLANATION} | 부연 설명 | "경쟁이 적은 곳에서 시작하세요" |
| {CTA_MESSAGE} | 행동 유도 | "도움이 됐다면 저장!" |
| {VISUAL_ELEMENT} | 시각 요소 | "lightbulb icon" |
