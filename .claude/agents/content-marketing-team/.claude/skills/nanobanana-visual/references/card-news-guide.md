# 카드뉴스 디자인 가이드

## 개요

인스타그램 카드뉴스는 스와이프를 통해 정보를 전달하는 멀티슬라이드 콘텐츠입니다.
한 장 한 장이 독립적이면서도 전체적으로 연결되어야 합니다.

## 기본 스펙

### 크기
- **권장**: 1080 x 1350px (4:5 세로형)
- **대안**: 1080 x 1080px (1:1 정사각형)

### 슬라이드 구성
- **최소**: 5장
- **권장**: 7장
- **최대**: 10장

## 슬라이드별 역할

### 슬라이드 1: 표지
**목적**: 스와이프 유도

**구성요소**:
- 메인 타이틀 (큰 글씨)
- 서브타이틀 (작은 글씨)
- 시각적 요소 (아이콘/일러스트)
- "스와이프 →" 또는 화살표

**예시 문구**:
- "알아두면 돈 되는 OOO"
- "OOO 하는 5가지 방법"
- "직장인이라면 꼭 알아야 할 OOO"

### 슬라이드 2-6: 본문
**목적**: 핵심 정보 전달

**구성요소**:
- 포인트 번호 (1, 2, 3...)
- 핵심 문구 (10-20자)
- 부연 설명 (30-50자)
- 관련 아이콘/일러스트

**원칙**:
- 1슬라이드 = 1포인트
- 너무 많은 텍스트 금지
- 시각적 여백 확보

### 마지막 슬라이드: CTA
**목적**: 행동 유도

**구성요소**:
- 요약 또는 마무리 문구
- CTA 버튼/문구
- 팔로우 유도
- 저장 유도

**예시 문구**:
- "저장해두고 나중에 보세요"
- "도움이 됐다면 팔로우"
- "친구에게 공유하기"

## 색상 가이드

### 배경색 추천
- **밝은 톤**: #FFFFFF, #F5F5F5, #FFF9E6
- **어두운 톤**: #1A1A2E, #16213E, #0F0F0F
- **포인트 톤**: 브랜드 컬러 활용

### 텍스트 색상
- 밝은 배경 → 어두운 텍스트 (#000000, #333333)
- 어두운 배경 → 밝은 텍스트 (#FFFFFF, #F0F0F0)

### 포인트 색상
- 강조할 숫자/키워드에 사용
- 브랜드 컬러 또는 보색 활용

## 타이포그래피

### 폰트 선택
- **제목**: 두꺼운 산세리프 (Bold, Black)
- **본문**: 중간 두께 산세리프 (Regular, Medium)

### 폰트 크기
- **메인 타이틀**: 60-80pt
- **서브 타이틀**: 30-40pt
- **본문**: 24-32pt

### 줄간격
- 1.3-1.5배

## 디자인 원칙

### 1. 일관성
- 모든 슬라이드에 동일한 스타일 적용
- 같은 폰트, 색상, 레이아웃 유지

### 2. 가독성
- 충분한 여백
- 높은 대비
- 큰 텍스트

### 3. 단순함
- 슬라이드당 요소 최소화
- 핵심 메시지에 집중
- 장식 요소 절제

### 4. 모바일 최적화
- 작은 화면에서도 잘 보이게
- 엄지로 스와이프하기 편하게
