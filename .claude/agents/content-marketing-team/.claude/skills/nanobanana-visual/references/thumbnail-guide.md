# 유튜브 썸네일 디자인 가이드

## 개요

썸네일은 영상의 첫인상입니다. 검색 결과와 추천 영상에서 클릭을 유도하는 가장 중요한 요소입니다.

## 기본 스펙

### 크기
- **해상도**: 1280 x 720px
- **비율**: 16:9
- **파일 형식**: PNG 또는 JPG
- **최대 용량**: 2MB

## 구성 요소

### 1. 텍스트
**특징**:
- 3-5단어 이내
- 큰 글씨 (화면의 30-50% 차지)
- 읽기 쉬운 폰트

**효과적인 문구 유형**:
- 숫자 포함: "3가지 방법", "월 100만원"
- 질문형: "왜 실패할까?"
- 이익 제시: "이것만 알면 끝"
- 충격/호기심: "아무도 안 알려준"

### 2. 인물
**활용법**:
- 표정이 드러나게 (놀람, 기쁨, 호기심)
- 눈이 카메라를 향하게
- 상반신 또는 얼굴 클로즈업

### 3. 배경
**유형**:
- 단색 배경 (선명함)
- 그라데이션 (세련됨)
- 흐릿한 사진 (컨텍스트)

### 4. 강조 요소
**종류**:
- 화살표
- 동그라미
- 밑줄
- 느낌표/물음표
- 이모지

## 색상 가이드

### 눈에 띄는 색상 조합
- **빨강 + 노랑**: 긴급함, 에너지
- **파랑 + 흰색**: 신뢰, 전문성
- **검정 + 노랑**: 경고, 주목
- **보라 + 핑크**: 트렌디, 창의적

### 피해야 할 조합
- 비슷한 명도의 색상
- 대비가 낮은 조합
- 너무 많은 색상 (3가지 이내)

## 타이포그래피

### 폰트 선택
- 두꺼운 산세리프 (Bold, Black)
- 가독성 높은 폰트
- 손글씨 폰트는 포인트로만

### 텍스트 효과
- 테두리(Stroke): 배경에서 분리
- 그림자: 입체감
- 하이라이트: 키워드 강조

## 레이아웃 패턴

### 패턴 1: 인물 중심
```
┌──────────────────────┐
│ [텍스트]             │
│        [인물]        │
│                      │
└──────────────────────┘
```

### 패턴 2: 텍스트 중심
```
┌──────────────────────┐
│                      │
│     [큰 텍스트]      │
│     [서브 텍스트]    │
└──────────────────────┘
```

### 패턴 3: 분할 구도
```
┌──────────┬───────────┐
│ [인물]   │ [텍스트]  │
│          │           │
└──────────┴───────────┘
```

### 패턴 4: Before/After
```
┌──────────┬───────────┐
│ Before   │  After    │
│  [이미지] │ [이미지]  │
└──────────┴───────────┘
```

## A/B 테스트 포인트

### 테스트할 요소
- 텍스트 문구
- 인물 표정
- 색상 조합
- 레이아웃

### 성과 지표
- 클릭률 (CTR)
- 노출 대비 클릭
- 시청 지속 시간

## 체크리스트

### 필수 확인
- [ ] 모바일에서 텍스트가 읽히는가? (작게 보여도)
- [ ] 3초 안에 내용을 파악할 수 있는가?
- [ ] 영상 내용과 일치하는가? (어그로 금지)
- [ ] 다른 영상 썸네일과 차별화되는가?

### 최적화 확인
- [ ] 색상 대비가 충분한가?
- [ ] 텍스트가 3-5단어 이내인가?
- [ ] 불필요한 요소가 없는가?
- [ ] 브랜드 일관성이 있는가?

## 금지 사항

- ❌ 클릭베이트 (내용과 다른 썸네일)
- ❌ 너무 많은 텍스트
- ❌ 저해상도 이미지
- ❌ 읽기 어려운 폰트
- ❌ 과도한 효과
