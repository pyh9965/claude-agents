# 쇼츠/릴스 플랫폼별 스펙 가이드

## 개요

각 플랫폼의 숏폼 콘텐츠 규격을 정리합니다.
이 문서를 참고하여 플랫폼에 최적화된 쇼츠를 제작합니다.

## 플랫폼별 스펙 비교

| 항목 | YouTube Shorts | Instagram Reels | TikTok |
|------|---------------|-----------------|--------|
| **비율** | 9:16 | 9:16 | 9:16 |
| **해상도** | 1080x1920 | 1080x1920 | 1080x1920 |
| **최대 길이** | 60초 | 90초 | 10분 |
| **권장 길이** | 15-60초 | 15-30초 | 15-60초 |
| **파일 형식** | MP4, WebM | MP4, MOV | MP4, MOV |
| **최대 용량** | - | 4GB | 287MB |
| **프레임레이트** | 24-60fps | 24-60fps | 24-60fps |

## YouTube Shorts 상세

### 기술 스펙
- **비율**: 9:16 (세로)
- **해상도**: 1080x1920px 권장
- **길이**: 15초-60초
- **파일 형식**: MP4, WebM
- **프레임레이트**: 24fps 이상

### 콘텐츠 가이드라인
- 세로 영상만 Shorts로 인식
- 60초 초과 시 일반 영상으로 업로드
- #Shorts 해시태그 권장
- 음악 라이브러리 사용 가능

### 알고리즘 포인트
- 첫 3초 중요 (시청 유지율)
- 끝까지 시청하면 재노출
- 반복 시청 유도가 효과적

## Instagram Reels 상세

### 기술 스펙
- **비율**: 9:16 (세로)
- **해상도**: 1080x1920px
- **길이**: 15초-90초
- **파일 형식**: MP4, MOV
- **최대 용량**: 4GB

### 콘텐츠 가이드라인
- 피드 미리보기: 1:1로 크롭됨
- 커버 이미지 설정 가능
- 트렌딩 오디오 활용 권장
- 캡션 2,200자 제한

### 알고리즘 포인트
- 오디오 트렌드 활용
- 첫 프레임이 중요 (피드 노출)
- 저장/공유 수 중요

## TikTok 상세

### 기술 스펙
- **비율**: 9:16 (세로)
- **해상도**: 1080x1920px
- **길이**: 15초-10분
- **파일 형식**: MP4, MOV
- **최대 용량**: 287MB (모바일), 500MB (웹)

### 콘텐츠 가이드라인
- 자체 편집 도구 활용 가능
- 듀엣/스티치 기능
- 트렌딩 사운드 중요
- 해시태그 전략 필수

### 알고리즘 포인트
- 완료율이 가장 중요
- For You Page 노출이 핵심
- 빠른 초반 후킹

## 공통 제작 가이드

### 안전 영역 (Safe Zone)

```
┌────────────────────────────┐
│ ← 150px → 제목 영역 ← 150px → │
│                              │
│    ┌──────────────────┐      │
│    │                  │      │
│    │   메인 콘텐츠    │      │
│    │   안전 영역      │      │
│    │                  │      │
│    └──────────────────┘      │
│                              │
│ ← 150px → UI 영역 ← 150px → │
└────────────────────────────┘
```

### 피해야 할 영역
- **상단 150px**: 프로필, 팔로우 버튼
- **하단 200px**: 좋아요/댓글 버튼, 설명
- **우측 100px**: 액션 버튼

### 자막 위치
- **권장**: 화면 중앙-하단
- **피해야 할 곳**: 하단 200px (UI와 겹침)
- **글자 크기**: 모바일에서 잘 보이게

## 자막 스타일 가이드

### 폰트
- 굵은 산세리프 권장
- 나눔고딕 Bold, Pretendard Bold

### 크기
- 본문: 40-50pt
- 강조: 60-80pt

### 색상
- 기본: 흰색 (#FFFFFF)
- 테두리: 검정 (#000000)
- 강조: 노랑 (#FFFF00) 또는 브랜드 컬러

### 위치
- 화면 중앙
- 하단에서 250px 이상 위

## 인코딩 설정

### ffmpeg 권장 설정
```bash
ffmpeg -i input.mp4 \
  -vf "scale=1080:1920:force_original_aspect_ratio=decrease,pad=1080:1920:(ow-iw)/2:(oh-ih)/2" \
  -c:v libx264 \
  -preset slow \
  -crf 18 \
  -c:a aac \
  -b:a 192k \
  -movflags +faststart \
  output.mp4
```

### 품질 설정
- **CRF**: 18-23 (낮을수록 고품질)
- **Preset**: slow 또는 medium
- **Audio**: 192k AAC
