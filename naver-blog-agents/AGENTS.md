# 글또 (Geulto) - AI 에이전트 팀

네이버 블로그 콘텐츠 제작을 위한 12인 전문가 팀

> **글또** = 글 + 또박또박
> 또박또박 정성스럽게 글을 써드리는 AI 에이전트 팀입니다.

## 팀 구성도

```
                        ┌─────────────┐
                        │  민준 팀장  │
                        │   (기획자)  │
                        └──────┬──────┘
                               │
          ┌────────────────────┼────────────────────┐
          │                    │                    │
          ▼                    ▼                    ▼
    ┌──────────┐        ┌───────────┐        ┌──────────┐
    │   수빈   │        │  작가팀   │        │  QA팀    │
    │ (리서처) │        │  (8명)    │        │  (2명)   │
    └──────────┘        └─────┬─────┘        └────┬─────┘
                              │                   │
        ┌─────────────────────┼─────────────────┐ │
        │    │    │    │    │    │    │    │    │ │
        ▼    ▼    ▼    ▼    ▼    ▼    ▼    ▼    ▼ ▼
      현우  지은  태현  하린  유진  민석  수아 예원맘 서연 준서
      (정보)(마케팅)(리뷰)(맛집)(여행)(테크)(라이프)(육아)(편집)(SEO)
```

## 워크플로우 파이프라인

```
Planning → Research → Writing → [Editing + SEO] → Formatting
   │          │          │            │              │
  민준       수빈      작가 1명    서연+준서       자동
 (25초)     (25초)     (40초)   (40초, 병렬)     (즉시)
```

| 단계 | 에이전트 | 설명 |
|-----|---------|------|
| 1. Planning | 민준 팀장 | 콘텐츠 전략, 작가 배정, 아웃라인 |
| 2. Research | 수빈 | 정보 수집, 팩트체크 |
| 3. Writing | 작가 1명 | 초안 작성 (Self-Reflection 포함) |
| 4. Editing | 서연 실장 | 교정, 톤 조정 |
| 5. SEO | 준서 | 키워드, 메타 최적화 |
| 6. Formatting | - | HTML/MD/JSON 변환 |

> **Note**: Editing과 SEO는 병렬로 실행되어 시간을 절약합니다.
