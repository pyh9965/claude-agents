=====================================================
구현된 CLI 명령어 (6개)
=====================================================

1. blog-image config
   - 설정 및 API 키 상태 확인
   - 옵션: 없음

2. blog-image collect <파일> [옵션]
   - 이미지 수집 및 배치
   - 옵션:
     -o, --output <디렉토리>
     -m, --model <모델>
     -p, --priority <우선순위>
     --no-optimize

3. blog-image generate -p "<프롬프트>" [옵션]
   - AI 이미지 생성
   - 옵션:
     -p, --prompt <텍스트> (필수)
     -t, --type <유형>
     -s, --style <스타일>
     -o, --output <파일>

4. blog-image analyze <파일> [옵션]
   - 콘텐츠 분석
   - 옵션:
     -o, --output <JSON파일>
     -m, --model <모델>

5. blog-image insert <파일> -i <디렉토리> [옵션]
   - 이미지 삽입
   - 옵션:
     -i, --images <디렉토리> (필수)
     -o, --output <파일>

6. blog-image pipeline <파일> [옵션]
   - 전체 파이프라인 실행
   - 옵션:
     -o, --output <디렉토리>

=====================================================
사용 예시
=====================================================

# 설정 확인
blog-image config

# 이미지 수집
blog-image collect examples/sample_blog_post.html -o output

# AI 이미지 생성
blog-image generate -p "맛있는 김치찌개" -t food_photo -s food

# 콘텐츠 분석
blog-image analyze examples/sample_blog_post.html -o reqs.json

# 이미지 삽입
blog-image insert content.html -i ./images -o result.html

# 전체 파이프라인
blog-image pipeline examples/sample_blog_post.html -o final
